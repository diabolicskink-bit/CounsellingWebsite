# Working market-research notes

This directory contains retained, non-canonical notes produced during delegated
market-segment-research tasks. Files may be rough, repetitive, incomplete,
internally inconsistent, based on non-comparable samples, or partly unverified.
Their presence does not make their contents reusable market evidence.

For new delegated work, use:

```text
<YYYY-MM-DD>--<segment-slug>--<operation-slug>[-<n>]/
  <subtask-slug>.md
```

Each subagent writes only to its assigned file. The parent research agent reads
all packets, checks sources and comparison rules, reconciles duplicates and
non-comparable evidence, assigns final source IDs, and integrates selected
material into an indexed dossier.

Outside that same continuing parent task, ignore these files unless the owner
explicitly requests their inspection. Do not use them as sources for SEO,
content, positioning, product work, later market research, or dossier refreshes.
The indexed market dossiers are the reusable authority.

Keep working files as ordinary Git-tracked project files. Agents must not
delete, move, rename, tidy, or overwrite them at task completion. The owner owns
their later review and deletion.
