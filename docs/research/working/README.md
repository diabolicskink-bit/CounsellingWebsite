# Working audience-research notes

This directory contains retained, non-canonical notes produced during delegated
audience-research tasks. Files may be rough, repetitive, incomplete, internally
inconsistent, or partly unverified. Their presence does not make their contents
approved research evidence.

For new delegated work, use:

```text
<YYYY-MM-DD>--<audience-slug>--<operation-slug>[-<n>]/
  <subtask-slug>.md
```

Each subagent writes only to its assigned file. The parent research agent reads
the packets, checks exact figures, quotations, or unusually consequential
claims, assigns final source IDs, and integrates selected material into an
indexed dossier.

Outside that same continuing parent task, ignore these files unless the owner
explicitly requests their inspection. Do not use them as sources for copy,
strategy, SEO, design, later audience research, or dossier refreshes. The
indexed audience dossiers are the reusable authority.

Keep working files as ordinary Git-tracked project files. Agents must not
delete, move, rename, tidy, or overwrite them at task completion. The owner owns
their later review and deletion.
